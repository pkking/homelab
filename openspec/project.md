# 项目概述

## 1. 项目简介

*   **项目名称:** 我的家庭实验室 (Homelab)
*   **一句话描述:** 一个用于管理和自动化我的家庭服务器的 GitOps 仓库。
*   **目标用户:** 项目所有者 / 家庭实验室爱好者
*   **解决的问题:** 自动化家庭服务器的配置和应用部署，降低手动管理的复杂性。

## 2. 技术栈

*   **前端:** 无
*   **后端:** 无
*   **数据库:** PostgreSQL
*   **基础设施/部署:** Kubernetes, ArgoCD, Helm, Kustomize
*   **关键依赖:** ArgoCD, Kubernetes

## 3. 开发规范

*   **代码风格:** yamllint
*   **版本控制:** Git
*   **测试策略:** 暂无
*   **提交信息规范:** Conventional Commits
